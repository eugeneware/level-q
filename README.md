# level-q

FIFO queueing for leveldb/levelup

[![build status](https://secure.travis-ci.org/eugeneware/level-q.png)](http://travis-ci.org/eugeneware/level-q)

## Installation

This module is installed via npm:

``` bash
$ npm install level-q
```

## Example Usage

``` js
var levelQ = require('level-q');
```
